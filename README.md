# AI-Hallucination-Benchmark

Project Link : https://colab.research.google.com/drive/1-5XpnLFK3WucQgcVhWauDxo2pOTj1MNp?usp=sharing

This project presents a structured benchmark to evaluate hallucination behavior in large language models (ChatGPT, Gemini, Claude). Using a controlled set of prompts, it measures performance through weighted accuracy (~86–100%), hallucination rate, refusal accuracy, and category-wise analysis across factual, citation, logic, and unanswerable queries. The study highlights differences in model reliability and confidence calibration, particularly in citation-based tasks.
